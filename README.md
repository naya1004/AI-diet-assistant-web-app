# 🌿 오늘의 건강식단 AI (Today's Healthy Meal AI)

현재 날짜와 계절을 기반으로 제철 식재료, 요리, 다이어트 메뉴, 운동을 추천하는 모바일 최적화 AI 식단 어시스턴트 웹앱입니다.

---

## 📋 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 파일명 | `diet-app.html` |
| 타입 | 단일 HTML 파일 (Vanilla JS) |
| 대상 | 한국 사용자, 모바일 최적화 |
| 화면 너비 | 최대 480px (모바일 중심) |
| 외부 의존성 | 없음 (CDN·라이브러리 미사용) |

---

## ✨ 주요 기능

### 1. 🏠 홈 — 오늘의 건강 브리핑
- 현재 날짜와 계절 자동 감지
- 오늘의 건강 메시지 출력
- 제철 식재료 TOP 3 요약
- 추천 요리 2가지 미리보기
- 오늘의 운동 추천 미리보기

### 2. 🥬 제철 식재료
- 현재 계절 기준 제철 식재료 목록
- 카테고리 필터: **채소 / 과일 / 수산물 / 곡물**
- 식재료별 제공 정보: 제철 이유, 영양성분, 건강 효능, 칼로리

### 3. 🍽️ 레시피
- 제철 식재료를 활용한 요리 추천
- 요리별 제공 정보: 주요 재료, 예상 칼로리, 조리시간, 난이도, 추천 이유

### 4. 🥗 다이어트 요리
- 카테고리 필터: **저탄수화물 / 고단백 / 저지방 / 간편식 / 도시락**
- 요리별 영양 정보: 칼로리, 단백질, 탄수화물, 지방
- 포만감 점수 시각화 (1~10점 바 그래프)

### 5. 🏃 운동 추천
- 현재 계절에 맞는 운동 목록
- 운동별 제공 정보: 운동 시간, 예상 칼로리 소모, 난이도, 추천 대상

### 6. 👤 내 프로필 — AI 맞춤 추천
- 입력 항목: 성별, 나이, 키, 현재 체중, 목표 체중, 활동 수준
- **Harris-Benedict 공식** 기반 BMR/TDEE 자동 계산
- 출력 결과: BMI 및 체중 상태, 하루 유지 칼로리, 목표 달성 권장 칼로리, 권장 단백질 섭취량, 맞춤 식단 및 운동 추천

## 🗂️ 계절별 데이터 구성

| 계절 | 월 | 대표 식재료 예시 | 추천 운동 예시 |
|------|----|-----------------|---------------|
| 봄 🌸 | 3~5월 | 냉이, 딸기, 주꾸미, 쑥 | 공원 걷기, 자전거 |
| 여름 🌞 | 6~8월 | 토마토, 블루베리, 장어, 오이, 수박 | 수영, 실내 사이클 |
| 가을 🍂 | 9~11월 | 고구마, 사과, 꽃게, 밤 | 등산, 조깅 |
| 겨울 ❄️ | 12~2월 | 시금치, 귤, 굴, 당근 | 홈트레이닝, 근력운동 |

## 🚀 실행 방법

```bash
# 별도 설치 없이 브라우저에서 바로 실행
diet-app.html 파일을 브라우저로 열기
```

또는 로컬 서버 사용 시:

```bash
# Python
python -m http.server 8000

# Node.js (npx)
npx serve .
```

## 📌 참고 사항

- 모든 추천 데이터는 **현재 날짜 기준 계절**에 따라 자동 변경됩니다.
